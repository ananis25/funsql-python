`funsql` is a python library to write compositional sql, and mostly a line by line port of the Julia library [FunSQL.jl](https://github.com/MechanicalRabbit/FunSQL.jl/). 

## Installation

Install this library using `pip`:

    $ pip install funsql-python

## Development

To contribute to this library, checkout the code in a new virtual enviroment. 

Now install the dependencies and test dependencies:

    pip install -e '.[test]'

To run the tests:

    pytest
